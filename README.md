# esp32_tft128x160

基于 ESP32-WROOM-32E + ST7735S 1.8寸 TFT彩屏项目。

## 硬件配置

| 组件 | 型号/参数 |
|------|----------|
| MCU | ESP32-WROOM-32E (4MB Flash) |
| 屏幕 | ST7735S 128x160 TFT (横屏显示，旋转3) |
| 背光 | D22 (HIGH有效) |

### 按钮引脚

| 按钮 | 引脚 | 功能 |
|------|------|------|
| K1 | D32 | 上/上一个城市 |
| K2 | D33 | 下/下一个城市 |
| K3 | D34 | 确定 |
| K4 | D35 | 返回/长按2秒重启 |

### SPI引脚

| 功能 | 引脚 |
|------|------|
| MISO | 19 |
| MOSI | 23 |
| SCLK | 18 |
| CS | 5 |
| DC | 16 |
| RST | 17 |

## 功能特性

### 1. 主菜单 (3项)

- **天气** - 默认启动页
- **WiFi信息** - 显示WiFi名称、IP、信号强度、NTP时间
- **WiFi配网** - AP网页配网模式

### 2. 天气功能

- **API**: `https://uapis.cn/api/v1/misc/weather?city=城市名` (免费无需key)

- **显示内容**:
  - 城市名 (2倍放大)
  - 天气描述 (2倍放大)
  - 温度 (大号显示)
  - 湿度
  - 风力
  - 风向
  - 日期时间 (右上角，每秒更新)
- **自动刷新**: 10分钟

### 3. WiFi配网

- **AP模式**: 热点名 `ESP32` (开放)
- **网页地址**: `http://192.168.4.1`
- **功能**:
  - 自动扫描附近WiFi
  - 支持手动输入WiFi名称
  - 配置保存在Flash (Preferences库)
- **默认配置**: `Misaka_Network15311` / `zxc456123`

### 4. 中文显示

- 自定义16x16点阵字库 (cn_font.h)
- 支持缩放显示 (scale参数)
- 70个汉字
- Python脚本 `gen_font.py` 生成字库

## 按钮逻辑

| 状态 | K1 | K2 | K3 | K4 |
|------|----|----|----|----|
| 菜单 | 上移 | 下移 | 确定 | 长按重启 |
| 天气 | 上一城市 | 下一城市 | - | 返回菜单 |
| WiFi信息 | - | - | - | 返回菜单 |
| WiFi配网 | - | - | - | 返回/长按清除配置 |

## 使用说明

### 首次使用

1. 烧录程序到ESP32
2. 设备会自动创建 `ESP32` 热点
3. 手机连接热点，浏览器访问 `http://192.168.4.1`
4. 选择或输入WiFi名称和密码
5. 保存后设备自动重启

### 日常使用

- 设备启动后自动连接WiFi，显示天气
- K1/K2 切换城市
- K3 进入菜单选择
- K4 返回上级菜单
- 菜单界面长按K4 2秒重启设备
- 配网界面长按K4 2秒清除WiFi配置

## 字库扩展

如需添加新汉字：

1. 编辑 `gen_font.py`，在 `chars` 变量中添加新汉字
2. 运行 `python gen_font.py` 生成新的字库
3. 重新烧录程序

## 注意事项

- 屏幕为横屏显示 (160x128)
- 中文字库使用16x16点阵，支持缩放
- 天气API免费无需key，但有请求频率限制
- WiFi配置保存在Flash，断电不丢失
